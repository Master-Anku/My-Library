repository အသစ်တည်ဆောက်ပုံ

Create a new repositoryကို နှိပ်ပါ
repo name [your name(apple)]
public ထားပေးပါ
Create repository ကို နှိပ်လိုက်ပါ။

See the image below.

repository အသစ်ဆို ဘယ်လို လုပ်ရမယ် ၊ ရှိပီးသာဆို ဘယ်လို လုပ်ရမယ်ဆိုတာကို အရင်ဆုံး ပြပေးပါလိမ့်မယ်။

ကိုယ်က အသစ်တည်ဆောက်မယ်ဆို ---
မိမိရဲ့ Terminal မှာ
git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin git@github.com:Master-Anku/hhh.git
git push -u origin main
