Git branch

Git branch ကို အသစ်တည်ဆောက်မယ်ဆို ပထမဆုံး Terminal မှာ ....

git branch dev

လက်ရှိ တည်ဆောက်ထားတဲ့ branch ကို ရောက်ဖို့က

git checkout dev

git branch နဲ့ ပြန်ကြည့်လိုက်ရင် devဆိုတဲ့ branchကို ရောက်နေမှာပါ။

git branch whit push

git add .

git commit -m "create dev branch

git push -u origin dev

အားလုံး အဆင်ပြေမယ်လို့ မျှော်လင့်လျက်.....
